# ربات تلگرام عکاسی عروس

ربات ثبت سفارش و نمایش پکیج‌های آلبوم/عکاسی عروس، با اطلاع‌رسانی خودکار به ادمین.

## امکانات
- نمایش پکیج‌ها با قیمت (`/packages`)
- ثبت سفارش گام‌به‌گام (`/order`): نام، شماره تماس، تاریخ مراسم
- ذخیره سفارش‌ها در دیتابیس محلی (SQLite)
- ارسال خودکار پیام اطلاع‌رسانی سفارش جدید به تلگرام خودت

## نکته‌ی خیلی مهم درباره‌ی ایران
تلگرام در ایران فیلتره. این یعنی:
- **کاربرانی که در ایران هستن**، برای استفاده از ربات باید از فیلترشکن/پروکسی روی گوشی یا سیستم خودشون استفاده کنن. این به کد ربات ربطی نداره و از دست ما خارجه.
- اما **سروری که این ربات رو روش اجرا می‌کنی، باید خارج از ایران باشه**، چون باید مستقیم و بدون مانع به `api.telegram.org` وصل بشه. اگه سرور رو داخل ایران بگیری، ربات اصلاً کار نمی‌کنه.

گزینه‌های مناسب برای هاست (ارزون و ساده):
- **Oracle Cloud** – لایه‌ی رایگان همیشگی داره (Always Free)
- **Hetzner** / **Contabo** – سرورهای ارزون آلمان
- **Railway** / **Render** – دیپلوی خیلی سریع، مناسب پروژه‌های کوچیک (پلن رایگان محدوده)

## مرحله ۱: نصب کتابخانه‌ها
```bash
pip install -r requirements.txt
```

## مرحله ۲: گرفتن توکن ربات
۱. در تلگرام به [@BotFather](https://t.me/BotFather) پیام بده
۲. دستور `/newbot` رو بزن و اسم و یوزرنیم دلخواه رو انتخاب کن
۳. توکنی که میده رو کپی کن

## مرحله ۳: گرفتن آیدی عددی خودت (برای دریافت اطلاع‌رسانی سفارش)
به ربات [@userinfobot](https://t.me/userinfobot) پیام بده، آیدی عددیت رو بهت میده.

## مرحله ۴: تنظیم متغیرها
دو راه داری:

**راه ساده (برای تست):** مستقیم در فایل `config.py` مقدار `BOT_TOKEN` و `ADMIN_CHAT_ID` رو جایگزین کن.

**راه امن‌تر (برای پروداکشن):** با متغیر محیطی تنظیم کن:
```bash
export BOT_TOKEN="123456:ABC-DEF..."
export ADMIN_CHAT_ID="987654321"
```

## مرحله ۵: ویرایش پکیج‌ها
پکیج‌ها، قیمت‌ها و توضیحاتشون رو در فایل `config.py` داخل لیست `PACKAGES` می‌تونی هر جور خواستی تغییر بدی (اضافه کردن پکیج جدید، حذف، تغییر قیمت و ...).

## مرحله ۶: اجرا
```bash
python3 bot.py
```
اگه پیام «ربات در حال اجراست...» رو دیدی، یعنی همه چیز درسته. حالا می‌تونی به ربات خودت در تلگرام پیام بدی و `/start` رو بزنی.

## اجرای دائمی روی سرور (تا با بستن ترمینال قطع نشه)
با `systemd` یا ابزار `pm2` / `screen` / `tmux` می‌تونی ربات رو به صورت دائمی روی سرور اجرا کنی. ساده‌ترین راه با `screen`:
```bash
screen -S mybot
python3 bot.py
# سپس Ctrl+A و بعد D رو بزن تا از صفحه خارج بشی، ربات همچنان در پس‌زمینه اجرا می‌مونه
```

## دیدن سفارش‌های ثبت‌شده
سفارش‌ها در فایل `orders.db` (SQLite) ذخیره میشن. می‌تونی با ابزارهایی مثل [DB Browser for SQLite](https://sqlitebrowser.org/) بازش کنی و لیست سفارش‌ها رو ببینی، یا از تابع `get_all_orders()` در `database.py` در یک اسکریپت کوچیک استفاده کنی.

## قابلیت‌های بعدی که می‌تونی اضافه کنی
- دکمه «سفارش‌های من» برای مشتری تا وضعیت سفارشش رو ببینه
- ارسال گالری نمونه‌کار (عکس‌ها) کنار هر پکیج
- اتصال به گوگل‌شیت به‌جای دیتابیس محلی برای مدیریت راحت‌تر سفارش‌ها
- افزودن درگاه پرداخت آنلاین (در صورت نیاز بعداً)
